NETSCAN v1
TCP Port Scanner + Webcam Fingerprint + Credential Lookup

NetScan v1 es una herramienta de reconocimiento de redes TCP de codigo abierto basada en tecnologia web. 
Combina tres capacidades principales en una unica interfaz de terminal visual: escaneo de puertos TCP, identificacion y huella 
digital de camaras IP, y consulta de credenciales predeterminadas por fabricante.
